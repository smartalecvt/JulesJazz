# JulesJazz

A set of custom fonts, templates, and resources for creating professional jazz charts in MuseScore.

## Overview

JulesJazz is a collection of enhancements for MuseScore designed to produce more traditional jazz notation. It includes custom SMuFL-compatible fonts, a lead sheet template, and supporting resources developed to improve the appearance of chord charts and other jazz notation.

The primary goal is not to reinvent MuseScore's engraving engine, but to refine its visual output so that finished charts more closely resemble professionally engraved jazz publications.

## Contents
* JulesJazz.otf – Custom music font.
* JulesJazzText.otf – Companion text font.
* Template – Ready-to-use MuseScore jazz leadsheet template.
* Additional resources – Supporting files used by the templates and fonts.

## Features
* Improved jazz notation appearance.
* Cleaner tempo markings.
* Professionally styled lead sheets.
* Templates configured for common jazz chart layouts.
* Compatible with MuseScore's SMuFL font system.

## Compatibility

Developed and tested with recent versions of MuseScore 4.

Because MuseScore occasionally changes the way it loads custom music fonts, installation locations may differ between releases. If a future update causes JulesJazz to stop appearing, simply reinstall the fonts and template according to the instructions below.

## Installation

* Quit MuseScore.
* Install the JulesJazz fonts according to the installation instructions for your operating system and MuseScore version.
* Copy any included templates into your MuseScore templates folder.
* Restart MuseScore.
* Open one of the included templates or select JulesJazz as your music font where appropriate.

## Notes

JulesJazz is intended for users who prefer a more traditional jazz engraving style. It makes a number of visual adjustments while remaining compatible with MuseScore's notation workflow.
