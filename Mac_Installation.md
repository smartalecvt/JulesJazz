# Mac Installation

I've tested this on a Mac Studio M1, running Tahoe 26.5.1, and MuseScore 4.7.3. Your mileage may vary, but there's nothing terribly complicated here, so I'm hopeful these files will be broadly usable. If you have notes on installation, feel free to let me know.

## The JulesJazz Font

You'll want download everything in the MusicFonts folder, and copy that entire folder directly into your ~/Documents/MuseScore4/ folder. You should wind up with a folder named ~/Documents/MuseScore4/MusicFonts/JulesJazz. And it should contain these 3 files: JulesJazz.otf, JulesJazzText.otf, and metadata.json. 

You'll also want to install JulesJazz.otf and JulesJazzText.otf as system fonts. You can double-click on them and add them via Font Book, or copy them into your font library, however you handle these sorts of things on your machine. 

# The Jazz Leadsheet

I've created a jazz Leadsheet template that fully utilizes the JulesJazz font. To install it, move the folder to replace the folder that already exists in /Applications/MuseScore 4.app/Contents/Resources/templates/05-Jazz. I've tried unsuccessfully to add this as a separate template, but my installation of MuseScore won't see it unless it's replacing the default jazz template that MuseScore ships with. You might want to back up the default jazz template before you overwrite it. 

Happy charting!
